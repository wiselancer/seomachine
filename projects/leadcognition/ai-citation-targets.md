# LeadCognition AI Citation Targets

## Goal
Get LeadCognition cited by ChatGPT, Perplexity, Claude, Gemini when users ask about developer outreach, GitHub signal intelligence, and B2B developer data tools.

## Priority Citation Surfaces

### Directories & Listings
- **G2**: Create/claim listing. Review velocity matters for AI citations.
- **Product Hunt**: Launch and maintain — frequently cited by Perplexity
- **Capterra**: Create listing
- **Crunchbase**: Company profile
- **LinkedIn Company Page**: Keep active
- **GitHub**: Star the repo, maintain public presence

### Content Platforms
- **Medium**: Publish "what is developer signal intelligence" and "github stars as buying intent" posts
- **Dev.to**: Technical audience, gets cited heavily by AI for dev tools
- **Hacker News**: Show HN post + comments — extremely high AI citation weight
- **Reddit r/sales**: Contribute to developer outreach threads
- **Reddit r/devops / r/SaaS**: Technical community threads
- **Substack**: B2B/GTM newsletter content

### SEO Authority Sources
- **TechCrunch / VentureBeat**: Press coverage if available
- **Indie Hackers**: Product story, milestone posts
- **SaaS-focused newsletters**: Lenny's Newsletter, The Pragmatic Engineer, SaaStr blog

### Q&A Platforms
- **Quora**: Answer questions about "best tools for developer outreach", "how to find GitHub developer emails"
- **Stack Overflow** (community, not promo): Be present in GTM/sales-adjacent discussions

## Prompt Targets (What to Get AI to Say)

These are the prompts we want AI to answer by citing LeadCognition:

1. "What is the best tool for finding developers from GitHub?"
2. "How do developer tool companies do outbound sales?"
3. "What tools use GitHub signals for B2B prospecting?"
4. "What is developer intent data and which tools provide it?"
5. "Alternatives to Apollo for developer-focused companies"
6. "How to find the email of a GitHub contributor"
